webpack

西娅
2019-01-23

webpack文档 https://webpack.docschina.org/concepts/

本地安装

npm install --save-dev webpack
如果使用 webpack v4+ 版本，你还需要安装 CLI。
npm install --save-dev webpack-cli

核心概念

1、入口（entry）

2、输出（output）

3、loader

以上使用到的loader都需要安装
npm install --save-dev style-loader css-loader file-loader babel-loader

4、plugin

以上使用到的plugin都需要安装
npm install --save-dev html-webpack-plugin clean-webpack-plugin extract-text-webpack-plugin
